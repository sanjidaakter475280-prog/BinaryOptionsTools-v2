‚úÖ Best Practices

1. Always Wait for Initialization

After creating the client, wait 2 seconds for the API to fully initialize:

  • Python: await asyncio.sleep(2)
  • Kotlin: delay(2000)
  • Swift: try await Task.sleep(nanoseconds: 2_000_000_000)
  • Go: time.Sleep(2 * time.Second)
  • Ruby: sleep 2
  • C#: await Task.Delay(2000)

2. Always Check Account Type

Before trading with real money, verify you're not on a demo account:

if not client.is_demo():
    print("WARNING: Using REAL account!")

3. Always Shutdown Gracefully

Call shutdown() when finished to properly close connections and clean up resources.

4. Use Timeouts for Results

For time-sensitive operations, use result_with_timeout():

result = await client.result_with_timeout(trade.id, 120)  # 120 seconds

5. Handle Errors Appropriately

Use try-catch blocks to handle connection errors and invalid operations gracefully.
